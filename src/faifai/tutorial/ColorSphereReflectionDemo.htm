Colour Plane Reflection Demo

Learning objectives: to describe the colour reflected off an object of different colours when different colours of light is shining upon it.

The object can now be any colour. A coloured object means that the colour is what would be reflected off the object. In other words, the colour is not absorbed by the object.

A object with a white surface means that the object looks white when it is shone upon by white light. A white object does not absorb any colour of light.

A red object will reflect red when shone upon by a white light. White light contains red component and only the red component is reflected. The other colour components are absorbed.

However, a red object will reflect nothing when shone upon by a green light. Green light contains no red component and so nothing is reflected.
